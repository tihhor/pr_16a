1. Установить django при помощи команды:
	
python -m pip install -r requirements.txt


2. Создать проект при помощи команды:

 название_проекта


3. Запустить проект на локальном сервере (для проверки работоспособности):

python manage.py runserver

После проверки работоспособности остановить командой Ctrl+C


4. Добавить в проект новое приложение при помощи команды:

python manage.py startapp название_приложения


5. Наполнить приложение всем необходимым функционалом (views, urls, templates, static и тд)


6. Запустить готовый проект:

python manage.py runserver
